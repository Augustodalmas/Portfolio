Registro/Login

Este projeto é uma aplicação de GUI (Interface Gráfica do Usuário) desenvolvida em Python usando a biblioteca Tkinter para criar um sistema de login. Ele inclui funcionalidades para registrar usuários, fazer login, editar informações de usuário e excluir usuários em um banco de dados SQLite. O sistema também permite criptografar senhas usando o algoritmo SHA-256 e exibe mensagens de sucesso ou erro durante a interação com o usuário.

A aplicação possui uma tela inicial com opções para registrar e fazer login de usuários, bem como uma função para mostrar informações do perfil do usuário após o login bem-sucedido.

*A aplicação utiliza um arquivo de configuração (config.json) para especificar o banco de dados SQLite a ser utilizado.
*A senha dos usuários é criptografada antes de ser armazenada no banco de dados.
*A aplicação usa mensagens de caixa de diálogo para fornecer feedback ao usuário.
*A interface gráfica é estilizada com cores de fundo e botões.
*O código contém funções para centralizar as janelas na tela do computador.

No geral, esse projeto se concentra em criar uma interface para interação com um banco de dados de login de usuário, tornando-o adequado para aprendizado e demonstração de conceitos de programação com Tkinter e banco de dados SQLite.
